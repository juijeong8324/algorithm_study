# 뱀

## 문제

- **'Dummy'**
  - 사과를 먹으면 뱀 길이가 늘어나고 벽이나 자기자신의 몸과 부딪히면 게임 끝
  - 뱀의 이동 규칙
    1. 몸길이를 늘려 머리를 다음칸에 위치
    2. 벽이나 자기자신의 몸과 부딪히면 게임 끝
    3. 이동한 칸에 사과 존재 시 그 칸에 있던 사과는 없어지고 꼬리는 움직이지 않음
    4. 사과가 없으면 걍 몸길이를 줄여서 꼬리가 위치한 칸을 비워준다. → 즉 몸길이는 변하지 않고 단순히 이동한거임

- **Input**
  - `N` = N x N 의 보드 크기 (2 <= N <= 100)
  - `K` = 사과의 개수 (0 <= K <= 100)
  - 사과의 행, 열 위치
    - K줄의 두 정수
    - 모두 다름
    - 1행 1열에는 사과 없음
  - `L` = 뱀의 방향 변환 횟수 (1 <= L <= 100)
  - `X C` = 게임 시작 시간의 X초가 끝난 뒤 방향 C
    - `X` = 10000 이하의 양의 정수, 오름차순임
    - `C` = 왼쪽 90도 ('L') or 오른쪽 90도 ('D') 방향 회전

- **Output**

  게임 종료 시간(초)
  - 뱀의 시작 위치는 (0,0) 길이는 1, 처음에 오른쪽 방향을 보고 있음

<br>
<br>

## Key point

- **Simulation**
  - 뱀의 머리 방향을 기준으로 좌/우 회전이 결정된다.
  - 시계 방향 또는 반시계 방향으로 방향 벡터를 업데이트하면 된다.
  - 단, 방향 전환은 해당 초의 이동이 완료된 후에 적용한다

<br>

- **Deque**
  - 뱀의 길이가 늘어날 때는 머리 좌표를 앞에 추가하고, 줄어들 때는 꼬리 좌표를 뒤에서 제거한다.
  - 몸통은 그대로 유지되므로, 양방향 삽입/삭제가 가능한 Deque 자료구조가 적합하다.

<br>

## Algorithm Approach

1. 초기화

```python
    # 방향 설정
    dx = [0, 1, 0, -1]
    dy = [1, 0, -1, 0] # 오른쪽, 아래, 왼쪽, 위 (시계 방향 순)

    # 초기값
    snake = deque([(1, 1)]) # 뱀의 좌표
    d = 0 # 초기 방향: 오른쪽
    time = 0

    # 뱀의 방향 전환 정보 저장
    for i in range(L):
        X, C = input().strip().split()
        X = int(X)
        turn[X] = C

```

- `dx`, `dy`는 오른쪽을 시작으로 시계 방향 순서로 정의한다. 순서에 주의할 것.
- 오른쪽 회전 시 인덱스 +1, 왼쪽 회전 시 -1로 업데이트한다.
- 방향 전환 정보를 dict에 저장하면 특정 시각에 O(1)로 접근할 수 있다.

<br>

2. 현재 방향으로 이동

```python
    time += 1
    # 이동
    x, y = snake[0] # 현재 머리 좌표
    x = x + dx[d]
    y = y + dy[d]
```

<br>

3. 종료 및 사과 처리

```python
    if x <= 0 or x > N or y <= 0 or y > N: # 벽
        break
    if board[x][y] == 1: # 사과가 있으면 먹고 꼬리 유지
        board[x][y] = 0
    elif (x, y) in snake: # 몸통 충돌
        break
    else:
        snake.pop() # 사과 없으면 꼬리 제거

    snake.appendleft((x, y)) # 새로운 머리 추가
```

- 머리는 항상 이동한다. 사과 여부에 따라 꼬리 제거 여부가 결정된다.

<br>

4. 방향 전환 처리

```python
if time in turn: # 방향 전환
    if turn[time] == 'L':
        d -= 1
        d = d % 4
    else:
        d += 1
        d = d % 4
```

방향 전환은 해당 초의 이동이 완료된 후에 적용한다.

## 추가

- **deque 초기화 시 튜플 주의**

```python
    deque((1, 2))    # [1, 2] → tuple을 순회해서 각 원소를 넣음
    deque([(1, 2)])  # [(1, 2)] → tuple 자체를 원소로 넣음
```

deque에 tuple 인자를 추가하고 싶으면 [(a, b)]로 해야 한다. `deque((1,2))`는 tuple을 list/deque로 변환하는 것과 같은 효과를 가진다.
