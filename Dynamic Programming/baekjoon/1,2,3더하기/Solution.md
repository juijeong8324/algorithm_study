## 문제 요약

정수 4를 1, 2, 3의 합으로 나타내는 방법은 총 7가지가 존재한다.

합을 나타낼 때 **_수를 1개 이상_** 사용해야 한다.

정수 n이 주어졌을 때 n을 1, 2, 3의 합으로 나타내는 방법의 수를 구하시오.

n은 양수고 11보다 작다!

## 문제 분석

n이 매우 작기 때문에 백트래킹도 가능함 혹은 10중 for문!

1. **백트래킹**

2개의 합부터 11개의 합(n이 11 이하 이므로)까지

각 개수에 따른 합이 주어지면 몇 개 선택 시 1,2,3 중 뭘 선택했는지 결정하면 된다.

```cpp
int select(int cnt, int k){
		if(cnt == k){ // k개의 합일 때
				// 합 return
		}

		for(int i = 1; i <= 3; i++){
				// 현재 합에 i를 더했을 때
				select(cnt+1, k);
				// 현재 합에서 i를 빼기
		}
}

main(){
		for(int i=1; i < N; i++){
				// 최대 N자리까지 선택 가능
				sum += select(0, i);
		}
}
```

그러나 10중 for문과 백트래킹은 n의 수가 작기 때문에 가능한 구현이고 N이 상당히 크면 DP가 아닌 다른 방법을 사용하기 어렵다. (겹치는 계산을 반복하는 것과 다름 없음)

1. **DP**

D[i] = i일 때 1,2,3 덧셈의 수

D[4] = 3일 때 1,2,3 덧셈의 수 , 이때 이전의 결과를 모두 알고 있다면..?

- 4보다 1 작은 합들에서 +1을 해주면 된다. → D[3]
- 4보다 2 작은 합들에서 +2를 해주면 된다. → D[2]
- 4보다 3 작은 합들에서 +3을 해주면 된다. → D[1]

즉 D[1] = 1, D[2] = 2(1+1, 2), D[3] = 4(3, 1+2, 2 + 1, 1+1+1)

D[4] = D[3] + D[2] +D[1]= 7임!

초기화는 d[1] d[2] d[3]일 듯 i에서 1,2,3 만큼 작은 수를 접근해야 하므로

## 코드

```cpp
#include <bits/stdc++.h>

using namespace std;

int d[12];
int N;
int T;

int main(void) {
    ios::sync_with_stdio(0);
    cin.tie(0);

    cin >> T;

		// 먼저 1부터 10까지 테이블을 다 계산한 후
    d[1] = 1; d[2] = 2; d[3] = 4;
    for (int i = 4; i <= 10; i++) {
        d[i] = (d[i - 1] + d[i - 2] + d[i - 1];
    }

		// 매번 n이 들어올 때마다 d[n]을 출력
    for (int i = 0; i < T; i++) {
        cin >> N;
        cout << d[N] <, "\n;
    }

}
```
